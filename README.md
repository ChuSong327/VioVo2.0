# VioVo


A Single Page Application that streams real-time most popular trending videos on YouTube and enables users to search and play selected videos.

[Live](http://viovo.herokuapp.com/)


### Installing
```javascript
yarn install
```
### Run Locally
```javascript
yarn start
```
### Deployment
The app is deployed to Heroku.

### Tech Stack
* React
* React-Router
* Redux
* Axios
* Material UI

#### Notice
API key is hidden.
